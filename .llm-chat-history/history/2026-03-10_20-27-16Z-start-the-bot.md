# AI Write-up Detector Bot

A comprehensive bot to detect AI-generated content (ChatGPT, Claude, Gemini, etc.) in documents and text.

## Features

### Detection Methods

1. **Perplexity Analysis** - Language model based detection
   - AI-generated text tends to have lower perplexity (more predictable)
   
2. **Burstiness Detection** - Sentence length variance analysis
   - Human writing has high variation in sentence lengths
   - AI text tends to be more uniform
   
3. **Vocabulary Richness** - Lexical diversity analysis
   - Measures Type-Token Ratio (TTR)
   - AI tends to have more repetitive vocabulary
   
4. **Word Repetition Analysis** - Pattern detection
   - Detects bigram and trigram repetition
   - AI often repeats certain phrases
   
5. **Punctuation Pattern Analysis**
   - AI tends to use uniform punctuation patterns
   
6. **Function Word Analysis**
   - Analyzes usage of articles, prepositions, pronouns
   
7. **Watermark Detection**
   - Detects known AI-specific phrases and patterns
   - Identifies GPT, Claude, and other AI patterns

## Installation

```bash
cd ai_writeup_detector_bot
pip install -r requirements.txt
```

## Usage

### Command Line

```bash
# Analyze a file
python run_ai_detector.py document.pdf

# Analyze a text file
python run_ai_detector.py sample.txt

# Analyze raw text
python run_ai_detector.py --text "Your text here"

# Skip report generation
python run_ai_detector.py file.txt --no-report

# Save results to JSON
python run_ai_detector.py file.txt --output results.json
```

### Python Code

```python
from ai_detector import detect_file, detect_text

# Detect AI content in a file
results = detect_file("document.pdf")

# Detect AI content in text
results = detect_text("Your text to analyze...")

# Access results
print(f"AI Probability: {results['ai_probability']}%")
print(f"Risk Level: {results['risk_level']}")
print(f"Verdict: {results['verdict']}")
```
